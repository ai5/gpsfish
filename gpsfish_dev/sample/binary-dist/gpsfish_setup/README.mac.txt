README
======

gpsfish.appアイコンをApplicationsアイコンにドラッグ＆ドロップしてインストールします。
/Applications/gpsfish.app/Contents/MacOS/gpsfish が実際のバイナリです。


To install gpsfish, drag & drop the "gpsfish.app" icon to the "Applications" icon.
/Applications/gpsfish.app/Contents/MacOS/gpsfish is the actual executable binary.
